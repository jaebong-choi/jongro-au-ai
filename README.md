# AU Study Guide — 호주 대학 진학 AI 진단 (학부·석사)

호주 학부(Bachelor)·석사(Master) 진학 경로를 진단하는 무료 정보 가이드 페이지입니다. 특정 유학원·기관과 무관하며, 수록된 학비·입학요건은 2026년 각 대학 공식 발표 기준입니다.

## 주요 기능

- **학부 / 석사 트랙 선택** — 히어로에서 진입 트랙을 선택하면 트랙별 맞춤 질문 진행
- **선호 지역 진단** — NSW(시드니) / VIC(멜버른) / QLD(브리즈번·골드코스트) / 기타 / 상관없음
- **세분화된 프로필 질문** — 영어 시험(IELTS/PTE/TOEFL) 점수대, 학부는 내신·수능, 석사는 학사 학점
- **지역 기반 추천** — 선호 지역 대학을 우선 정렬하고 "선호 지역 일치" 배지 표시
- **패스웨이 로직** — Direct Entry / Diploma Transfer / Foundation / GE / Graduate Certificate 등 자동 판정
- **대학 공식 코스 링크** — 추천 프로그램별 각 대학 공식 코스 페이지로 바로 연결
- **결과 링크 공유** — 진단 결과가 URL 해시에 인코딩되어, 링크를 복사·전달하면 동일한 결과 페이지가 바로 열림

## 기술 스택

- 단일 `index.html` (정적 페이지) — Tailwind CDN, Font Awesome, Noto Sans KR
- GitHub Pages 배포

## 면책

본 페이지는 정보 제공 목적이며, 입학요건·학비·비자 요건은 수시로 변경됩니다. 지원 전 반드시 각 대학 공식 입학처와 [호주 정부 공식 유학 포털](https://www.studyaustralia.gov.au/)에서 최신 정보를 확인하세요.
